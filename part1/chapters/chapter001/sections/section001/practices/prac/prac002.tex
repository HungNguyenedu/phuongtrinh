\begin{baitap}
 Giải phương trình: $ x +\sqrt{x^2 + x + 2 } =3  $
\begin{loigiai1}
Phương trình đã cho tương đương: $ \sqrt{x^2 + x +2 } = 3 - x $ \\
$ \Leftrightarrow 
\begin{cases}
    3-x \geq 0 \\
    x^2 + x + 2 = (3-x )^2 
\end{cases} $ \\
$ \Leftrightarrow
\begin{cases}
    x \leq 3 \\
    x^2 + x + 2 = 9 -6x + x^2
\end{cases} $ \\
$ \Leftrightarrow 
\begin{cases}
    x \leq 3 \\
    x=1
\end{cases} $ 
$ \Leftrightarrow x=1 $ \\
Kết luận: Vậy phương trình có nghiệm duy nhất: $ x=1 $.
\end{loigiai1}

\end{baitap}
