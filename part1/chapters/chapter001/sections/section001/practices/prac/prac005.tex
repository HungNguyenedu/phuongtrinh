\begin{baitap}
 Giải phương trình: $ \sqrt{x+8 } - \sqrt{x } = \sqrt{x + 3}    $
\begin{loigiai1}
Điều kiện: $ x \geq 0 $. \\
Với điều kiện trên phương trình tương đương: $ \sqrt{x+8 } = \sqrt{x } + \sqrt{x + 3}  $ \\
$ \Leftrightarrow x+8 = 2x +3 + 2 \sqrt{x(x+3 ) } $ \\
$ \Leftrightarrow 5-x = 2 \sqrt{x^2 + 3x } $ \\
$ \Leftrightarrow \begin{cases}
    x \leq 5 \\
    25 -10x + x^2 = 4x^2 + 12x
\end{cases} $ \\
$ \Leftrightarrow \begin{cases}
    x \leq 5 \\
    3x^2 +22x -25 =0
\end{cases} $ \\
$ \Leftrightarrow \begin{cases}
    x \leq 5 \\
    \left[
        \begin{array}{l}
            x =1 \\
            x= -\dfrac{25 }{3 }  
        \end{array}
    \right. 
\end{cases} $ \\
$ \Leftrightarrow \left[
    \begin{array}{l}
        x =1 \\
        x=-\dfrac{25 }{3 }  
    \end{array}
\right. $ \\
Kết hợp điều kiện ta được $x=1$ là nghiệm duy nhất của phương trình.
\end{loigiai1}

\end{baitap}
